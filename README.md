## IN55
Re-upload of an old school project.
It is a tool that simulates the parametric construction of 3D objects and more precisely of gems. It was developed in C++ and OpenGL 3

## How to move

|                           |           **Input**           |
|---------------------------|-------------------------------|
|       Move Forward        |               Z               |
|       Move Backward       |               S               |
|       Move Right          |               D               |
|       Move Left           |               Q               |
|       Move Up             |               E               |
|       Move Down           |               A               |
|Put the light where you are|               L               |
